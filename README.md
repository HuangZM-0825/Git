# Git 練習倉庫

這個倉庫包含了一系列的 Jupyter Notebook 文件和相關資源，旨在教導如何使用 Git 進行版本控制。透過這些練習，您將學會如何使用 Git 來管理和協作開發專案，並掌握分支策略和工作流程的設計與實作。

## 目標

這個倉庫的主要目的是幫助您熟悉 Git 的基本概念和操作，並通過實際的練習來掌握如何在 Git 環境中進行版本控制和協作開發。您將學會如何：

1. 使用 Git 進行版本控制。
2. 創建和修改 Git 項目。
3. 使用 Git 來共用作業和協作開發。
4. 透過分支與合併來管理程式碼變更。
5. 設計及實作分支策略和工作流程。

## 練習內容

### 1. 使用 Git 進行版本控制簡介
這部分包含了 Git 的基本介紹、如何創建和修改 Git 項目、使用 Git 來共用作業，以及透過分支與合併來編輯程式碼的教學。

### 2. 描述原始檔控制的目的和使用方式
這部分說明了原始檔控制系統如何協助管理檔案，並包含一個引導式專案，教導如何使用 GitHub 來管理檔案更新。

### 3. 設計及實作分支策略和工作流程
這部分介紹了如何設計及實作分支策略和工作流程，幫助您在實際專案中更有效地管理和協作開發。

## 如何開始

1. 克隆這個倉庫到本地端：
    ```sh
    git clone <倉庫網址>
    ```
2. 按照每個 Notebook 的教學步驟進行操作。

## 提示

- 確保已安裝 Git 並配置好 GitHub 帳號。
- 使用 VSCode 或其他支援 Git 的編輯器來進行開發和版本控制。
- 在練習過程中，請隨時參考 Git 的官方文件和社群資源。

這個練習題目將幫助你熟悉如何在 Git 環境中進行版本控制和協作開發，並掌握分支策略和工作流程的設計與實作。祝你練習愉快！
